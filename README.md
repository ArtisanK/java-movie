
# 영화 예매
---
## 기능 요구사항

1. 영화 기본 정보로 여러 항목이 주어진다.
1. 영화는 한번에 여러 영화를 예매할 수 있어야 한다.
1. 예메가 가능한 상태이면 결제를 진행한다.
1. 예매가 불가능한 경우 그 이유를 보여주고, 다시 예매할 수 있도록 해야한다.
1. 결제 유형에 따라 할인율이 달리지고, 포인트를 사용할 수 있다.
1. 최종 예매 금액을 보여준다.
